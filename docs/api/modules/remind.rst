:mod:`Remind` --- Set reminders
===============================

Commands
--------

.. cmdoption:: .at <time>

    Remind at a specific time.
    
    Example: ``.at 20:45EST Do your homework!``
    
.. cmdoption:: .in <time>

    Remind after an amount of time.
    
    Example: ``.in 1week5d2h1m Go fuck yourself``
