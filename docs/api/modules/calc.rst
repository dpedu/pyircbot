:mod:`Calc` --- IRC fact game
=============================

Commands
--------

.. cmdoption:: calc

     Returns a random fact

.. cmdoption:: calc <something> = <some definition>

    Add a fact

.. cmdoption:: calc <something> =

    Delete a fact

.. cmdoption:: match <keyword>
