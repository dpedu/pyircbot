pyircbot3
=========
**A modular python IRC bot**

Quick start
-----------

* Install: `python3 setup.py install`
* Configure: `cd examples ; vim config.yml data/config/Services.yml`
* Run: `./run-example.sh` or `pyircbot -c config.yml`
